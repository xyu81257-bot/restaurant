美味餐厅点餐管理系统
项目简介
本项目为《软件工程》课程小组作业，基于 Spring Boot + Vue3 + MySQL 前后端分离架构，面向中小型线下餐饮门店，实现顾客自助扫码点餐、后台菜品 / 桌台 / 库存 / 订单管理、营业数据可视化统计一体化轻量级餐饮管理平台。系统区分顾客端、管理员端两大角色，无需第三方平台入驻，无抽成、数据本地可控，适配夫妻店、中小型餐馆日常经营数字化需求。
技术栈
后端
核心框架：Spring Boot 2.7
ORM 持久层：MyBatis-Plus
数据库：MySQL 8.0
身份认证：JWT 无状态登录
构建工具：Maven
工具：Lombok、Transactional 事务控制
前端
框架：Vue3 + Vue Router + Pinia
UI 组件：Element Plus
网络请求：Axios
数据可视化：ECharts
开发工具：VS Code
核心功能
顾客端
用户注册、登录、身份鉴权
菜品分类浏览、菜品详情查看
购物车增删改、数量调整
选择桌台下单、库存自动校验
订单支付、历史订单查询、未支付订单取消
管理端
菜品管理：新增 / 编辑 / 上下架 / 删除菜品
分类管理：菜品分类增删改查
桌台管理：桌台新增、状态可视化、结账清台自动完结订单
库存管理：库存阈值预警、库存增减记录
订单管理：全量订单查询、状态修改
用户管理：管理员账号维护
经营统计：今日 / 月度营收、热销菜品排行、营业额趋势图表
